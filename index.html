Canvas Help

Use the APIOps Cycles canvas creator via the web interface or as an npm module or CLI. You can also create your own canvases if you introduce your own JSON schema. You can also use AI to fill in canvases for you, and then import them to the UI or export as SVG / PNG for viewing

  • Open: Import JSON or select locale & canvas. Tip: you can first export an empty canvas
  • Create: Double-click canvas to create new stickies.
  • Edit: Double-click sticky note to edit.
  • Change Color: Click note, pick color.
  • Remove: Right-click & confirm.
  • Edit later: Export as JSON (on your computer).
  • Use elsewhere: Export as SVG or PNG for slides/web.
  • CLI: npm i canvascreator
  • Localize: Make a pull request or issue in the GitHub repo
