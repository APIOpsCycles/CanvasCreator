APIOps Cycles Canvas Creator

For instructions how to use the APIOps Cycles canvases, check the method and the canvas instructions at apiopscycles.com.

If you are insterested in the canvases as JSON files or helping to localize the canvases or use and contribute this code, check the Github repository

For similar, but more integrated commercial tool, you can refer to one of our partners Qriar Labs. For consulting or training services on how to use the canvases,check our partner page for more information.

  • Open a canvas: Select the locale (language) and then the canvas.
  • Create sticky notes: Import from JSON file or double-click to create or edit sticky notes.
  • Change note color: Click sticky note and then pick color. Selected color will be used for new stickies, too.
  • Remove note: Right click (on touch devices long press) and confirm to remove a sticky note.
  • Save your content: Export as JSON. For security and simplicity reasons, everything is done in your browser and not sent to any server.
  • Export as image: You can export the whole canvas and content as SVG (vector image) to use in slides, Miro / Mural, print or web.
