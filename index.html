APIOps Cycles Canvas Creator

For instructionsb> how to use the APIOps Cycles canvases, check the method and the canvas instructions at apiopscycles.com.

If you are insterested in the canvases as JSON files or helping to localize the canvases or use and contribute this code, check the Github repository

  • Open a canvas: Select the locale (language) and then the canvas.
  • Create sticky notes: Import from JSON file or double-click left mouse button to create or edit sticky notes.
  • Change note color: Left click sticky note and then pick color. Selected color will be used for new stickies, too.
  • Remove note: Right click and confirm to remove a sticky note.
  • Save your content: Export as JSON. For security and simplicity reasons, everything is done in your browser and not sent to any server.
  • Export as image: You can export the whole canvas and content as SVG (vector image) to use in slides, Miro / Mural, print or web.
